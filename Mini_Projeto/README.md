**GET Pokemon Name of Pokemon Number**
```
https://pokeapi.co/api/v2/pokemon/145
- forms.name
```

**GET Moves and URL of Pokemon Number**
```
https://pokeapi.co/api/v2/pokemon/145
- moves.move.name / url
```

**GET Power and Accuracy of Move Number**
```
https://pokeapi.co/api/v2/move/87
- power / accuracy
```
